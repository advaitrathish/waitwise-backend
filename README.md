## Backend Status: Completed

This repository contains the completed backend for the WaitWise project.

### Features implemented
- Django REST Framework based APIs
- CRUD operations for core entities
- JWT authentication (login)
- Protected APIs using `IsAuthenticated`
- Token-based authorization tested via API client

### Tech Stack
- Django
- Django REST Framework
- SimpleJWT
- SQLite / PostgreSQL (as applicable)

### Status
Backend development is complete and stable.  
Frontend is maintained in a **separate repository**.
